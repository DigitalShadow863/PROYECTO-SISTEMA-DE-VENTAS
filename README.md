# Sistema de Ventas CasaCell

Proyecto desarrollado en Python para la gestión y control de ventas de accesorios para celulares en la tienda CasaCell.

El sistema permite registrar ventas, aplicar descuentos, seleccionar métodos de pago, generar reportes, almacenar información en archivos y realizar pruebas con registros simulados. Su objetivo es reemplazar el proceso manual realizado en cuadernos por una solución digital más eficiente y segura.
